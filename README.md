# express-api

### raspbery pi terminal commands
<br/> 

### Clone Repo
```shell
git clone https://github.com/prichardsondev/express-api.git
```
<br/> 

### Change Directory
```shell
cd express-api
```
<br/> 

### Open express-api in VSCode
```shell
code .
```
<br/> 

### Download software/dependencies updates and install
```shell
sudo apt update && sudo apt upgrade
```
<br/> 

### install nodejs
```shell
curl -fsSL https://deb.nodesource.com/setup_19.x | sudo -E bash - &&\
sudo apt-get install -y nodejs
```

### install express-api dependencies
```shell
npm i
```
<br/>

### run api
```shell
npm start
```
<br/>

### Web browser
```shell
localhost:5000
```
```shell
localhost:5000/add?v1=10&v2=11
```
